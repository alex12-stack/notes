Веб-приложение для управления заметками

Репозиторий для выполнения индивидуального проекта по разработке прикладного веб-приложения для работы с заметками.
Приложение предназначено для пользователей, которые хотят удобно хранить, структурировать и совместно редактировать текстовые заметки.

Пользователь может:
  создавать, редактировать и удалять заметки
  группировать заметки по папкам
  делать заметки приватными или публичными
  делиться доступом к заметкам или папкам с другими пользователями
  назначать права просмотра или редактирования
  выполнять поиск по заметкам и папкам

Хранимые данные:
  Пользователи (логин, пароль, дата регистрации)
  Заметки (название, текст заметки, дата создания, дата изменения, тип доступа, владелец заметки)
  Папки (название, дата создания, владелец)
  Доступ (пользователь, заметка или папка, тип доступа)

Основные действия:
  Работа с заметками/папками (создание, удаление, редактирование, управление доступом)
  Поиск и сортировка (поиск заметок/папок по названию, сортировка по названию/типу/дате создания)

Структура базы данных:
  users (пользователи)
  notes (заметки)
  folders (папки)
  note_access (доступ к заметкам)
