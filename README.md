# ---
Проект представляет собой веб-приложение для заметок. Пользователь сможет добавлять,удалять,изменять заметки, заметки могут быть приватные и публичные. Будет возможность группировать заметки по папкам, делать сортировку по типу заметки/названию/дате создания. Приватные заметки будут доступны только самому пользователю, публичные заметки будут видеть те, у кого есть доступ, также можно будет выдать права редактора, чтобы одну заметку/папку могли изменять сразу несколько человек. Будет реализован поиск по названию заметки/папки
